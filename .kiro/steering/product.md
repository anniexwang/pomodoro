# Product Overview

This is a React Native mobile application built with Expo Router, designed to run on iOS, Android, and web platforms. The app follows a tab-based navigation structure with a home screen and explore screen.

## Key Features
- Cross-platform compatibility (iOS, Android, Web)
- Dark/light theme support with automatic system detection
- Tab-based navigation with haptic feedback
- Modal presentation support
- Responsive design with platform-specific adaptations

## Target Platforms
- iOS (with tablet support)
- Android (with adaptive icons and edge-to-edge display)
- Web (static output with favicon support)

The application uses Expo's new architecture and experimental features like typed routes and React Compiler for enhanced performance and developer experience.
